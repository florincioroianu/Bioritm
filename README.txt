    BIORITM:
    Bioritm este o aplicatie-joc ce ruleaza pe sistemul de operare Android. Scopul acestei aplicatii este de a arata
o corelatie intre bioritmul zilnic si o culoare specifica. Ea este adresata persoanelor interesate de ciclurile bioritmice
ce vor sa isi pastreze un echilibru in viata.
    Pe langa partea ce calculeaza culoarea specifica, in aplicatie mai este implementat un mic joc ce foloseste
ca tema culoarea respectiva avand rol de a bine dispune utilizatorul.
    Aplicatia a fost construita de la 0 in Unity 2019.2.0b2, script-ul scris in Microsoft Visual Studio 2017 iar cele
cateva imagini folosite in Adobe Photoshop CS 5.
